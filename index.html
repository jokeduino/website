What is JokeDuino.

JokeDuino is an Andruino based device for jokes and pranks.

How it works

It can sense the sorrounding evironment and generate weird sounds in right moment in time.
See how to make a mosquito prank with Jokeduino ;-)

How to get one?

Currently the only way to have one is to make it yourself.

If you love hacking with hardware as we do you can build one yourself by following documentation on github.
